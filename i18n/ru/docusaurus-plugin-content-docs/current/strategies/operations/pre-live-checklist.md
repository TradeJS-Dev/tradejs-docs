title: Чек-лист перед live-запуском
---

Этот список стоит пройти до включения `--makeOrders`.

## Качество стратегии

- Есть покрытие бэктестами по разным рыночным фазам.
- В оценке учтены комиссии/проскальзывание.
- Нет явной переоптимизации под один короткий период.

## Готовность runtime

- Конфиг стратегии загружен и валиден.
- AI/ML policy настроены осознанно.
- Логика TP/SL проверена.

## Готовность инфраструктуры

- `yarn doctor` проходит по обязательным проверкам.
- Redis/Postgres стабильны.
- ML gRPC доступен, если требуется.

## Безопасность запуска

- Выставлены лимиты экспозиции.
- Есть rollback-план.
- Настроены алерты на критичные ошибки.
