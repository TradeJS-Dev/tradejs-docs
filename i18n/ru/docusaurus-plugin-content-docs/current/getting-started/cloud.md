Вариант работы через managed cloud сейчас находится в разработке.

## Что это значит сейчас

- публичный cloud-процесс еще не зафиксирован как стабильный продуктовый сценарий
- структура инфраструктуры, требования к безопасности и схема обновлений еще уточняются
- документация по cloud будет расширяться по мере стабилизации

## Текущий endpoint для тестирования

- `https://tradejs.dev/`

## Что использовать в проде сейчас

Для production рекомендуется использовать:

- `getting-started/self-hosted`
- `getting-started/local` (для разработки и проверки)
